# ./code

This directory contains almost all of the code of the application. The CLI can be launched from here by invoking `node .` or `npm run cli`.

| file | description |
| - | - |
| app.js    | main routine and also the CLI launcher |
| libraries | custom helpers and library code used in the app |
| loaders   | ETL scripts for loading third party data |
| queries   | SQL query generators |
| utilities | command line utilities |
